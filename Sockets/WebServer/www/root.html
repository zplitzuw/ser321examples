You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply? -- returns two integers multiplied ex: /multiply?num1=8&num2=5
  • /github? -- returns the ID, repo name, and repo owner ex: /github?query=users/amehlhase316/repos
  • /triangle_area? -- returns the area of a triangle ex: /triangle_area?base=5&height=3
  • /random_number_generator? -- returns a random number in between two numbers ex: /random_number_generator?low_num=2&high_num=90
